For gmap API key please follow the below steps:

  1. login to gmail account
  2. go to the following link https://developers.google.com/maps/documentation/javascript/get-api-key
  3. click on get a key button
  4. create project name and get api key
  5. copy that api key and replace on old api key which present in index file see following code (Replace api key with key parameter). 
    <script type="text/javascript" src="https://maps.googleapis.com/maps/api/js?libraries=places&
	key=AIzaSyAsdzx6REuLqvQJC9VffvqM_84hUlKbWyM"></script>
